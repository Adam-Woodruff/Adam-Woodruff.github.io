A level Computer science project

For my A level computer science project I intend to create an encryption tool that allows the user to encrypt plain text using ether the RSA algorithm or a one time pad. In the final program I plan to create a graphical interface using pythons built in tkinter library to make the program more intuitive. As of now i have programmed the one both encryption algorithms as well as some extra features for example a menu, a database to save keys and a check-sum to ensure the keys are the same. To finish my project I must finish the interface, tidy my code and finish my write up for the project.
